
## How to Add Images

1. **Modeling Images**: Place in `images/modeling/` folder
2. **Building Images**: Place in `images/building/` folder
3. **Scripting Screenshots**: Place in `images/scripting/` folder
4. **UI/UX Designs**: Place in `images/ui-ux/` folder

The portfolio will automatically display placeholder images if your actual images are not uploaded yet.

## Updating Content

Edit these files:
- `index.html` - For text content and structure
- `css/style.css` - For styling and colors
- `js/script.js` - For interactive features

## Technologies Used
- HTML5, CSS3, JavaScript
- Font Awesome Icons
- Google Fonts (Inter)
- GitHub Pages for hosting
